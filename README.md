# nextjs-react-query-zod-cms

## Required Installed Software

1. NodeJS 20 and up

## How to run this??

1. ``$ cd <root path of nextjs-react-query-zod-cms>``
2. ``$ yarn install``
3. ``$ cp .env.example .env``
4. Change the values in the .env as needed.
5. ``$ yarn dev``

___

## Logging in the CMS
